# Basic-Chatbot-for-flight-bookings-
A simple flight booking chatbot web app built with HTML, CSS, JavaScript, and Flask. It features a responsive UI, login/signup modals, and an interactive chatbot that provides flight info for destinations like Goa, Bangalore, London, and USA using a backend API.

This project is a simple web-based flight booking chatbot built using HTML, CSS, JavaScript, and Flask (Python). It provides an interactive interface where users can communicate with a chatbot to get basic flight information.

The frontend includes a responsive navigation bar, sections like Home, About, Registration, and Contact, along with login and sign-in modals. A floating chatbot button opens a chat interface where users can type queries. JavaScript handles sending messages to the backend and dynamically updating the chat.

📄 Frontend file:

The backend is developed using Flask, which processes user input and returns responses based on predefined keywords. The chatbot can respond to greetings, provide help, and give approximate flight prices for destinations like Goa, Bangalore, London, and the USA.

🔄 Working
User opens the website and interacts with the chatbot
Message is sent to the Flask server via /chat API
Backend processes input and sends a response
Response is displayed in the chat interface

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Communication: Fetch API
